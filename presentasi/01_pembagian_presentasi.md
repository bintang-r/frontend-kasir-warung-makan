## 📢 Agenda Alur Presentasi Kelompok

Untuk mendapatkan nilai terbaik dan presentasi yang dinamis, alur presentasi disarankan berjalan sebagai berikut:

1. **Pendahuluan**: Latar belakang aplikasi Rumah Makan Siantar Minang Jaya.
2. **Arsitektur & API (Oleh Anda)**: Menjelaskan arsitektur client-server dan bagaimana data backend mengalir ke frontend.
3. **Sisi Pengguna / Customer (Oleh Teman 1)**: Menunjukkan bagaimana pelanggan memesan menu melalui HP, memindai QR meja, atau melacak pesanan.
4. **Sisi Back-office / Admin (Oleh Teman 2)**: Menunjukkan bagaimana pemilik restoran mengelola menu, promo, memantau grafik penjualan, dan data user.
5. **Sisi Operasional / Kasir & Dapur (Oleh Teman 3)**: Menunjukkan bagaimana dapur memproses makanan (Kanban board) dan kasir menerima pembayaran (POS System) secara real-time.
